# Homes & Gardens Planet 🏗️

A fully responsive construction and interior design company website built with HTML, CSS and JavaScript.

## Live Features
- 🌙 Dark and light theme toggle
- 💬 WhatsApp form integration for direct client messaging
- 📱 Fully responsive design for mobile, tablet and desktop
- ✨ Scroll animations on counter highlights
- 🍔 Mobile hamburger menu with smooth open/close
- 🖼️ Project gallery with hover effects
- 📧 Contact form with floating label animation
- 6 dedicated service sub pages

## Pages
- Home page with hero, about, services, gallery and contact sections
- Architecture & Design Build
- Premium Decorative Painting
- Wall Skimming (Screeding)
- Custom Paint Production
- Project Management
- Consultancies

✅ Complete

- Google Fonts (Poppins)
